# JavaScript Project

A collection of JavaScript programs, mini-projects, and practice exercises created while learning JavaScript fundamentals. This repository demonstrates the use of core JavaScript concepts through simple and interactive examples.

## 📌 Features

- Beginner-friendly JavaScript programs
- DOM Manipulation examples
- Event Handling
- Functions and Objects
- Arrays and Strings
- Loops and Conditional Statements
- Basic Form Validation
- Browser Object Model (BOM)
- Navigator Object examples
- Interactive mini projects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/nityajhajharia/javascript.project.git
```

### 2. Open the project

Navigate to the project folder.

```bash
cd javascript.project
```

### 3. Run

Simply open any HTML file in your browser.

No installation or dependencies are required.
